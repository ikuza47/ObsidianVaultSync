> [!SUMMARY] Кратко
> **Основа**: Рывок-удар, Кулаки/нож, Казнь, Отталкивание
> **Философия**: Патроны ограничены — используй окружение и стелс

---

Поскольку патроны ограничены, игрок должен освоить бесшумные и бесплатные способы устранения врагов.

### Принципы ближнего боя

```
┌─────────────────────────────────────────────────────────┐
│  ЭКОНОМИЯ ПАТРОНОВ                                      │
└─────────────────────────────────────────────────────────┘

Удар → Удар → Удар → Казнь = 4 секунды, 0 патронов
Рывок → Отбрасывание → Пропасть = 2 секунды, 0 патронов
Стелс → Казнь = 1 секунда, 0 патронов, бесшумно
```

---

## Основные механики ближнего боя

| Способ | Описание | Когда использовать | Урон |
|--------|----------|-------------------|------|
| **Рывок-удар** | Игрок делает рывок в сторону врага, нанося урон | Против одиночных целей, в узких коридорах | Средний |
| **Нож/кулаки** | Базовая атака ближнего боя (3 удара в секунду) | Когда нужно бесшумно, против слабых врагов | Низкий |
| **Казнь** | Мгновенное убийство раненого врага (кнопка взаимодействия) | Добивание, экономия патронов | Мгновенная смерть |
| **Отталкивание** | Сильный удар, отбрасывающий врага | Чтобы сбросить врага в пропасть или ловушку | Зависит от окружения |

---

## Окружение как оружие

| Элемент | Использование | Эффект | Примечание |
|---------|---------------|--------|------------|
| **Шипы** | Заманить врага на шипы | Мгновенная смерть или урон | Требует планирования |
| **Пропасти** | Отбросить врага рывком | Мгновенная смерть | Самый быстрый способ |
| **Взрывные бочки** | Выстрелить/ударить бочку | Урон по площади | Привлекает других врагов |
| **Падающие объекты** | Подрубить верёвку, толкнуть ящик | Убийство без шума | Требует наблюдения |
| **Электрические лужи** | Заманить врага в воду | Урон + оглушение | Комбо с другими атаками |
| **Двери-ловушки** | Захлопнуть дверь на враге | Урон + временное оглушение | Для побега |

## Комбо-система ближнего боя

### Базовые комбо

```
┌─────────────────────────────────────────────────────────┐
│  КОМБО БЛИЖНЕГО БОЯ                                     │
└─────────────────────────────────────────────────────────┘

Базовое:
  Удар → Удар → Удар → Казнь
  4 секунды | 1 враг | 0 патронов

Быстрое:
  Рывок → Отбрасывание → Пропасть
  2 секунды | 1 враг | 0 патронов

Тихое:
  Приседание → Подход сзади → Казнь
  1 секунда | 1 враг | 0 патронов | Бесшумно
```

### Продвинутые комбо

```
Комбо с окружением:
  Отталкивание → Взрывная бочка → Урон по площади
  3 секунды | 2-3 врага | 0 патронов

Комбо со стелсом:
  Тень → Убийство из засады → Казнь
  2 секунды | 1 враг | 0 патронов | Бесшумно

Комбо с рывком:
  Рывок-удар → Добивание → Казнь
  3 секунды | 1 враг | 0 патронов
```
