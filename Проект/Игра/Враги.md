## Поведение врагов

### States (Состояния)

```
┌─────────────────────────────────────────────────────────┐
│  ИИ ВРАГОВ                                              │
└─────────────────────────────────────────────────────────┘

1. ПАТРУЛИРОВАНИЕ
   - Перемещение по заданному маршруту
   - Обнаружение игрока (звук, видимость)

2. ТРЕВОГА
   - При обнаружении тела или шума
   - Вызов подкрепления
   - Поиск источника

3. ПОИСК
   - Обыск зоны, где видели игрока
   - Длится 10-30 секунд
   - Возврат к патрулированию

4. БОЙ
   - Атака игрока
   - Использование укрытий
   - Отступление при низком HP

5. ОТСТУПЛЕНИЕ
   - Слабые враги бегут при низком здоровье
   - Поиск помощи
```

### Групповая тактика

| Тактика | Описание |
|---------|----------|
| **Окружение** | Враги пытаются зайти с флангов |
| **Подавление огнём** | Стрелки прикрывают ближников |
| **Координация** | Поддерживающие лечат других |
| **Засада** | Враги прячутся и ждут игрока |
