## Дизайн уровней

### Принципы дизайна

```
┌─────────────────────────────────────────────────────────┐
│  ДИЗАЙН УРОВНЕЙ                                         │
└─────────────────────────────────────────────────────────┘

1. НЕЛИНЕЙНОСТЬ
   - Несколько путей прохождения
   - Выбор тактики (стелс / агрессия)

2. СЕКРЕТЫ
   - Скрытые комнаты
   - Коллекционные предметы
   - Карточки навыков

3. СОКРАЩЕНИЯ
   - Открываемые пути
   - Быстрое перемещение
   - Ярлыки к боссам

4. ЧЕКПОИНТЫ
   - Только в торговых постах
   - Сохранение прогресса
   - Нет сохранений в бою
```

### Структура уровня

```
┌─────────────────────────────────────────────────────────┐
│  ПРИМЕР СТРУКТУРЫ УРОВНЯ                                │
└─────────────────────────────────────────────────────────┘

    ┌─────┐
    │ СТАРТ │
    └──┬──┘
       │
    ┌──▼──┐     ┌──────────┐
    │ Путь А │────│ Секрет 1 │
    └──┬──┘     └──────────┘
       │
    ┌──▼──┐     ┌──────────┐

    └─────┘
```

## Секреты и коллекционные предметы

### Типы секретов

| Тип | Описание | Награда |
|-----|----------|---------|
| **Скрытые комнаты** | За фальшивыми стенами, в вентиляции | Карточки, оружие |
| **Коллекционные предметы** | Разбросаны по уровням | Кристаллы, уникальные карточки |
| **Временные испытания** | Пройти за время | Бонусные монеты |
| **Бесшумное прохождение** | Без тревоги | Редкие карточки |

### Размещение секретов

```
┌─────────────────────────────────────────────────────────┐
│  РАЗМЕЩЕНИЕ СЕКРЕТОВ                                    │
└─────────────────────────────────────────────────────────┘

1. ЗА ФАЛЬШИВЫМИ СТЕНАМИ
   - Визуальная подсказка (отличный цвет)
   - Нужно ударить/выстрелить

2. НА ВЕРХНИХ ПЛАТФОРМАХ
   - Требуется двойной прыжок
   - Карточка или аптечка

3. В ТЁМНЫХ УГЛАХ
   - Требуется фонарик/внимание
   - Патроны или монеты

4. ПОСЛЕ ГОЛОВОЛОМКИ
   - Открыть дверь, активировать механизм
   - Редкая карточка
```
