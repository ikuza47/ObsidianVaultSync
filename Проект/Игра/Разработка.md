## Фаза 1 — Прототип ядра

**Цель**: Рабочий прототип с базовыми механиками

### Задачи

- [ ] Базовое перемещение (бег, прыжок, рывок)
- [ ] Рывок-удар по врагам
- [ ] Система здоровья врагов
- [ ] Подбор оружия с врагов
- [ ] Система магазинов (без перезарядки)
- [ ] Тестовая комната с врагами

### Критерии готовности

```
✅ Игрок может бегать, прыгать, делать рывок-удар
✅ Враги получают урон и умирают
✅ Оружие подбирается с врагов
✅ Патроны не перезаряжаются
```

**Время**: 2-4 недели

---

## Фаза 2 — Боевая система

**Цель**: Полная боевая система

### Задачи

- [ ] Все виды ближних атак (нож, казнь, отталкивание)
- [ ] 3 типа оружия (пистолет, револьвер, дробовик)
- [ ] Инвентарь и отображение боезапаса
- [ ] Простейшие враги (патрульный, стрелок)
- [ ] Ловушки окружения (шипы, пропасти)

### Критерии готовности

```
✅ 4 типа ближних атак работают
✅ 3 типа оружия с разным балансом
✅ HUD отображает здоровье, патроны
✅ 2 типа врагов с ИИ
✅ Окружение убивает врагов
```

**Время**: 3-5 недель

---

## Фаза 3 — Система карточек

**Цель**: Рабочая система карточек

### Задачи

- [ ] Базовая система карточек (5 обычных)
- [ ] Механика стакирования
- [ ] Механика убывающей эффективности
- [ ] Инвентарь карточек (слоты)
- [ ] Подбор карточек с врагов
- [ ] Торговец с карточками

### Критерии готовности

```
✅ 5+ карточек работают
✅ Стакирование линейное
✅ Убывающая эффективность
✅ Слоты карточек (5-15)
✅ Дроп с врагов
✅ Торговец покупает/продаёт
```

**Время**: 3-4 недели

---

## Фаза 4 — Terminal UI

**Цель**: ASCII-интерфейс

### Задачи

- [ ] ASCII-рамки и границы
- [ ] Библиотека иконок (20+)
- [ ] Генератор карточек
- [ ] HUD в стиле терминала
- [ ] Анимации (печатание, мерцание)
- [ ] Инвентарь и магазин (UI)

### Критерии готовности

```
✅ Все рамки ASCII
✅ 20+ иконок для карточек
✅ Карточки генерируются
✅ HUD отображает всё
✅ Анимации работают
✅ Инвентарь и магазин (UI)
```

**Время**: 4-6 недель

---

## Фаза 5 — Контент

**Цель**: Полный контент

### Задачи

- [ ] Все типы карточек (обычные, редкие, эпические, легендарные)
- [ ] 5-6 типов врагов
- [ ] 2-3 типа локаций
- [ ] Звуковые эффекты
- [ ] Визуальные эффекты карточек

### Критерии готовности

```
✅ 20+ карточек в каталоге
✅ 6 типов врагов с ИИ
✅ 3 типа локаций
✅ Звуки оружия, шагов, окружения
✅ Эффекты для карточек
```

**Время**: 6-8 недель

---

## Фаза 6 — Полировка

**Цель**: Готовый продукт

### Задачи

- [ ] Визуальные эффекты (кровь, частицы, освещение)
- [ ] Баланс оружия, врагов, карточек
- [ ] Оптимизация
- [ ] Баг-фиксинг
- [ ] UI/UX улучшения

### Критерии готовности

```
✅ Частицы, кровь, освещение
✅ Сбалансированный геймплей
✅ 60 FPS на целевых системах
✅ Критические баги исправлены
✅ Удобный интерфейс
```

**Время**: 4-6 недель
