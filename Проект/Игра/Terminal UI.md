> [!SUMMARY] Кратко
> **Стиль**: Терминал/консоль с ASCII-графикой
> **Шрифт**: Моноширинный
> **Цвета**: Зелёный (основной), красный (опасность), жёлтый (предупреждение)

> [!attention]
> Представленный в документации вид  [[TUI]] всего лишь пример, он не будет являться итоговым вариантом

---

Весь интерфейс выполнен в стиле **терминала/консоли** с использованием **ASCII-графики**. Это создаёт ощущение работы с древней компьютерной системой выживания.

### Стилевые принципы

| Принцип | Описание |
|---------|----------|
| **Моноширинный шрифт** | Все символы равной ширины (как в консоли) |
| **ASCII-символы** | Использование символов для графики (─│┌┐└┘├┤┬┴┼█░▒▓) |
| **Цветной текст** | Разные цвета для категорий (красный = урон, зелёный = здоровье) |
| **Минимализм** | Только необходимая информация |
| **Анимация символов** | Мерцание, печатание текста, сканирование |

### Цветовая схема интерфейса

```
┌────────────────────────────────────────┐
│  Основной текст: #00FF00 (зелёный)     │
│  Опасность: #FF0000 (красный)          │
│  Предупреждение: #FFFF00 (жёлтый)      │
│  Информация: #00FFFF (голубой)         │
│  Карточки: #FF00FF (фиолетовый)        │
│  Неактивное: #666666 (серый)           │
└────────────────────────────────────────┘
```

## Алгоритм генерации ASCII-карточек

```
┌─────────────────────────────────────────────────────────┐
│  АЛГОРИТМ ГЕНЕРАЦИИ ASCII-КАРТОЧКИ                      │
└─────────────────────────────────────────────────────────┘

1. ОПРЕДЕЛЕНИЕ ПАРАМЕТРОВ
   ├── Тип карточки (урон, защита, скорость, здоровье)
   ├── Редкость (обычная, редкая, эпическая, легендарная)
   ├── Название
   ├── Эффект
   ├── Иконка (ASCII-арт)
   └── Текущее количество стаков

2. ВЫБОР ASCII-ИКОНКИ
   │
   ├── Кулак → Ближний бой
   ├── Пистолет → Огнестрельное оружие
   ├── Щит → Защита/броня
   ├── Ботинок → Скорость/перемещение
   ├── Сердце → Здоровье/регенерация
   ├── Глаз → Крит/точность
   ├── Череп → Уникальные/легендарные
   └── Молния → Скорость атаки/рывок

3. ОТРИСОВКА РАМКИ
   │
   ├── Обычная:    ┌─────────────┐
   ├── Редкая:     ╔═════════════╗
   ├── Эпическая:  ╔═════════════╗ + по углам
   └── Легендарная:╔═════════════╗ + индикатор

4. РАЗМЕЩЕНИЕ КОНТЕНТА
   │
   ├── Верх: ASCII-иконка (по центру)
   ├── Середина: Название карточки
   ├── Описание: Текст эффекта
   ├── Статы: Стаки и итоговый эффект
   └── Низ: Кнопки действий [E] Взять [X] Выбросить

5. ЦВЕТОВАЯ СХЕМА (по редкости)
   │
   ├── Обычная:    Белый/серый текст
   ├── Редкая:     Зелёный текст
   ├── Эпическая:  Фиолетовый текст
   └── Легендарная: Оранжевый/красный текст + анимация

6. АНИМАЦИИ (опционально)
   │
   ├── Мерцание рамки для легендарных
   ├── Печатание текста при открытии
   └── Пульсация иконки
```
