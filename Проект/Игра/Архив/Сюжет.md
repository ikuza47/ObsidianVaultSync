

> [!SUMMARY] Кратко
> **Сеттинг**: Постапокалипсис, подземный комплекс
> **Подача**: Экологичное повествование, записки, диалоги
> **Герой**: Одиночка, пытающийся выбраться

---

## 🌍 Концепция мира

> Мрачный постапокалиптический мир после неизвестной катастрофы. Ресурсы крайне ограничены, особенно боеприпасы. Выжившие сражаются за каждый патрон, а оружие — ценнее золота.

> Главный герой — одиночка, пытающийся выбраться из огромного подземного комплекса, кишащего врагами. На пути встречаются редкие торговцы — такие же выжившие, торгующие найденными припасами и карточками навыков.

> **Карточки** — загадочные артефакты прошлого, содержащие зашифрованные знания о боевых техниках и улучшениях. Никто не знает их происхождения, но их сила очевидна.

## 🎭 Подача истории

### Экологичное повествование

История через окружение (трупы, записки, разрушения):

```
┌─────────────────────────────────────────────────────────┐
│  ПРИМЕРЫ ЭКОЛОГИЧНОГО ПОВЕСТВОВАНИЯ                     │
└─────────────────────────────────────────────────────────┘

1. ТРУПЫ В ПОЗАХ
   - Солдат сжимает фотографию семьи
   - Учёный тянется к терминалу
   - Самоубийство в углу

2. РАЗРУШЕНИЯ
   - Следы борьбы (кровь, гильзы)
   - Взломанные двери
   - Сожжённые документы

3. ГРАФФИТИ
   - Предупреждения на стенах
   - «ОНИ ЗДЕСЬ»
   - «БЕГИ ПОКА МОЖЕШЬ»
```

### Записки выживших

Найденные документы с историями других:

```
┌─────────────────────────────────────────────────────────┐
│  ЗАПИСКА #17                                            │
└─────────────────────────────────────────────────────────┘

«День 47. Патроны кончились. Старик умер ночью.
Они всё ещё ходят по коридорам. Я слышу их шаги.

Карточки... они дают силу, но я не понимаю, откуда
они взялись. Нашёл одну в теле солдата. Она
светилась...

Если кто-то найдёт это — не доверяй торговцам.
Они знают больше, чем говорят.»

                                            — А.
```

### Диалоги с торговцами

Редкие моменты «безопасности», намёки на сюжет:

```
┌────────────────────────────────────────────────────────────────┐
│     ╔══════════════════════════════════════════════════════╗   │
│     ║  СТАРОЙ:                                             ║   │
│     ║  «Ты не первый, кто ищет выход.                      ║   │
│     ║   Но первый, кто дошёл так далеко...                 ║   │
│     ║   Карточки... они зовут тебя, да?                    ║   │
│     ║   Они все зовут...»                                  ║   │
│     ╚══════════════════════════════════════════════════════╝   │
└────────────────────────────────────────────────────────────────┘
```

### Минимум катсцен

История рассказывается через геймплей:

- **Нет пауз в игре** для сюжета
- **Всплывающие сообщения** при находках
- **Аудиодневники** (опционально)
- **Финальная сцена** только в конце

## 🔮 Тайны и загадки

### Основные вопросы

```
┌─────────────────────────────────────────────────────────┐
│  ТАЙНЫ СЮЖЕТА                                           │
└─────────────────────────────────────────────────────────┘

1. ОТКУДА ВЗЯЛИСЬ КАРТОЧКИ?
   - Древняя цивилизация?
   - Эксперимент учёных?
   - Инопланетное происхождение?

2. КТО ВРАГИ?
   - Мутировавшие выжившие?
   - Роботы охраны?
   - Что-то другое?

3. ЧТО СЛУЧИЛОСЬ С МИРОМ?
   - Война?
   - Эпидемия?
   - Эксперимент вышел из-под контроля?

4. ПОЧЕМУ ТОРГОВЦЫ ЗНАЮТ БОЛЬШЕ?
   - Они создали карточки?
   - Они заперты здесь давно?
   - Они тоже выжившие?
```
