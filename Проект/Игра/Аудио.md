### Компоненты

| Элемент | Описание | Примеры |
|---------|----------|---------|
| **Музыка** | Мрачный эмбиент, индастриал, дарк-электроника | Боевая тема, стелс-тема |
| **Звуковые эффекты** | Выстрелы, шаги, удары, окружение | Пистолет, дробовик, рывок |
| **Атмосфера** | Фоновые звуки | Вентиляция, эхо, шум |
| **Озвучка** | Минимальная | Стоны, фразы при уроне |

---

### Музыкальные темы

#### По ситуациям

| Ситуация | Музыка | Описание |
|----------|--------|----------|
| **Исследование** | Мрачный эмбиент | Тихая, фоновая музыка |
| **Бой** | Индастриал / Дарк-электроника | Динамичная, нарастающая |
| **Стелс** | Минималистичный эмбиент | Напряжённая, тихая |
| **Торговый пост** | Спокойный эмбиент | Безопасность, отдых |
| **Босс** | Оркестр + электроника | Эпичная, интенсивная |

### Динамическая музыка

```
┌─────────────────────────────────────────────────────────┐
│  ДИНАМИЧЕСКАЯ МУЗЫКА                                    │
└─────────────────────────────────────────────────────────┘

1. НАРАСТАНИЕ ПРИ ОБНАРУЖЕНИИ
   - Тихая → Громкая
   - Медленная → Быстрая
   - 2-3 секунды переход

2. ЗАТИХАНИЕ В СТЕЛСЕ
   - Громкая → Тихая
   - Когда враги потеряли игрока
   - 5-10 секунд переход

3. СИГНАЛЫ ТРЕВОГИ
   - Звуковой сигнал при обнаружении
   - Уникальный для каждого типа врага
```

## Звуковые эффекты

### Оружие

| Оружие | Звук | Описание |
|--------|------|----------|
| **Пистолет** | Громкий выстрел, эхо в помещениях | Стандартный звук выстрела |
| **Револьвер** | Громче, с раскатом | Мощный звук |
| **Дробовик** | Очень громкий, дробный звук | Несколько выстрелов сразу |
| **ПП** | Серии выстрелов, быстро | Автоматический звук |
| **Винтовка** | Точный, с эхом | Дальнобойный звук |

### Ближний бой

| Действие | Звук | Описание |
|----------|------|----------|
| **Рывок-удар** | Свист + удар | Быстрый звук |
| **Нож/кулаки** | Удар плоти | Глухой звук |
| **Казнь** | Хруст + стон | Мгновенная смерть |
| **Отталкивание** | Удар + шаг | Сильный толчок |

### Окружение как оружие

| Элемент | Звук | Описание |
|---------|------|----------|
| **Шипы** | Лязг металла + хруст | Смертельная ловушка |
| **Взрывные бочки** | Взрыв + осколки | Громкий хлопок |
| **Электричество** | Треск + крик | Опасная зона |
| **Падающие объекты** | Удар + пыль | Тяжёлый предмет |

## Шаги

### По поверхностям

| Поверхность | Звук | Громкость |
|-------------|------|-----------|
| **Металл** | Лязг, стук | Громко |
| **Бетон** | Глухой стук | Средне |
| **Вода** | Хлюпанье, брызги | Средне |
| **Дерево** | Скрип, стук | Тихо |
| **Трава/земля** | Шуршание | Очень тихо |
| **Ковёр/ткань** | Мягкий звук | Бесшумно |

### По режимам перемещения

| Режим | Изменение звука |
|-------|-----------------|
| **Бег** | Громче, чаще |
| **Ходьба** | Стандартно |
| **Приседание** | Тише, реже |
| **Стелс** | Минимально |

## Атмосферные звуки

### По локациям

| Локация | Звуки |
|---------|-------|
| **Заброшенные коридоры** | Капли воды, скрип металла, далёкие шаги |
| **Подземные помещения** | Эхо, ветер, обвалы |
| **Промышленные зоны** | Станки, вентиляция, гудение |
| **Лаборатории** | Оборудование, сигналы, шипение |
| **Торговые посты** | Тихая музыка, шаги торговца |

### Временные эффекты

| Время | Изменения |
|-------|-----------|
| **Ночь** | Больше звуков врагов, тише фон |
| **День** | Стандартные звуки |

## Озвучка

### Игрок

| Ситуация | Фразы |
|----------|-------|
| **Получение урона** | «Аргх!», «Чёрт...», кряхтение |
| **Смерть** | Крик, падение |
| **Подбор карточки** | «Хм...», «Полезно...» |
| **Вход в торговый пост** | Вздох облегчения |

### Враги

| Тип | Звуки |
|-----|-------|
| **Патрульный** | Шаги, бормотание |
| **Стрелок** | Команды, выстрелы |
| **Бронник** | Тяжёлые шаги, рычание |
| **Берсерк** | Крики, рёв |
| **Босс** | Уникальные фразы, рёв |

### Торговцы

| Ситуация | Фразы |
|----------|-------|
| **Приветствие** | «Привет, странник...», «Вижу, ты ищешь припасы...» |
| **Покупка** | «Удачный выбор», «Хорошая сделка» |
| **Продажа** | «Не густо, но сойдёт», «Запомню тебя» |
| **Прощание** | «Возвращайся», «Удачи в глубине» |

## Аудио-фидбек

| Событие | Звук | Описание |
|---------|------|----------|
| **Громкость выстрелов** | Привлекает врагов в радиусе | Механика обнаружения |
| **Шаги** | Разные звуки для разных поверхностей | Металл, бетон, вода |
| **Эхо** | В больших помещениях | Пространственный звук |
| **Приглушение** | За стенами звуки тише | Реалистичность |
| **Динамическая музыка** | Нарастает при обнаружении, затихает в стелсе | Напряжение |
| **Звук подбора карточки** | Уникальный звук для каждой редкости | Обычные → Легендарные |

### Индикаторы звуком

```
┌─────────────────────────────────────────────────────────┐
│  ЗВУКОВЫЕ ИНДИКАТОРЫ                                    │
└─────────────────────────────────────────────────────────┘

1. ПОДБОР КАРТОЧКИ
   - Обычная: Тихий «пинг»
   - Редкая: Более высокий «пинг»
   - Эпическая: Аккорд
   - Легендарная: Уникальный звук + музыка

2. ВРАГИ РЯДОМ
   - Тихие шаги за стеной
   - Бормотание врагов
   - Лязг оружия

3. ОПАСНОСТЬ
   - Музыка нарастает
   - Звуки становятся громче
   - Сигнал тревоги
```
