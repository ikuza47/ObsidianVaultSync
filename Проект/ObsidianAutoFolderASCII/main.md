>[!Info] Находится на этапе идеи

Плагин для Obsidian, который будет автоматически создавать ASCII рисунок расположения файлов в vault
Плагин должен имеет следующие настройки:

| Опция          | Параметр                                                                                                                                                                              |
| -------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Ссылка на файл | **Вставка** - Ссылается на файл с использованием функционала Obsidian ```[[Название]]``` **Прямая ссылка** - Ссылается на файл используя путь с корневой папки ```[Название](Путь)``` |
|                |                                                                                                                                                                                       |
