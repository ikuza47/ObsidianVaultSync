> [!SUMMARY] Кратко
> **Определение**: Интерфейс с графическими элементами
> **Элементы**: Окна, иконки, меню, кнопки

---

**GUI** (Graphical User Interface) — система взаимодействия человека с компьютером, где информация представлена в виде графических элементов: окна, иконки, меню, кнопки, списки и т.д.

---

## Элементы GUI

| Элемент | Описание |
|---------|----------|
| **Окна** | Контейнеры для контента |
| **Иконки** | Визуальные представления действий |
| **Меню** | Списки опций и команд |
| **Кнопки** | Элементы для действий |
| **Поля ввода** | Ввод текста и данных |
| **Списки** | Перечисление элементов |

---

## Преимущества

- Интуитивно понятный интерфейс
- Визуальная обратная связь
- Удобство для новичков
- Стандартизированные элементы

---

## Применение в играх

| Тип | Описание |
|-----|----------|
| **HUD** | Отображение здоровья, патронов |
| **Меню** | Настройки, инвентарь |
| **Диалоги** | Взаимодействие с NPC |
| **Карты** | Навигация по миру |
