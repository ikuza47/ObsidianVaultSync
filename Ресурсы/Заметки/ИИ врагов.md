> [!SUMMARY] Кратко
> **Определение**: Искусственный интеллект противников
> **Компоненты**: Восприятие, Принятие решений, Действия
> **Типы**: Патрулирование, Поиск, Бой

---

**ИИ врагов** (Artificial Intelligence) — система поведения компьютерных противников, включающая восприятие окружения, принятие решений и выполнение действий.

---

## Архитектура

### Восприятие

**Чувства врага:**
- **Зрение** — поле зрения (FOV), дальность, линия видимости
- **Слух** — радиус слышимости, источники шума
- **Память** — запоминание последней позиции игрока

### Принятие решений

**Методы:**
- **Конечный автомат (FSM)** — переключение между состояниями
- **Дерево решений** — иерархический выбор действий
- **Дерево поведения (Behavior Tree)** — гибкая система приоритетов

### Действия

**Типы:**
- Движение (навигация, преследование)
- Атака (ближняя, дальняя)
- Групповая тактика (окружение, подавление)

---

## Конечный автомат (FSM)

### Состояния

| Состояние | Описание |
|-----------|----------|
| **Патрулирование** | Движение по заданному маршруту |
| **Подозрение** | Проверка источника шума |
| **Тревога** | Подготовка к бою |
| **Бой** | Атака игрока |
| **Поиск** | Поиск потерянного игрока |
| **Отступление** | Бегство при низком HP |

### Переходы

Между состояниями происходят переходы по триггерам (увидел игрока, услышал шум, таймер).

---

## Восприятие

### Зрение

**Параметры:**
- Угол обзора (FOV)
- Дальность видимости
- Линия видимости (LOS)

### Слух

**Параметры:**
- Радиус слышимости
- Уровень шума действий
- Приоритет источников

### Память

**Хранение:**
- Последняя известная позиция игрока
- Время с последнего обнаружения
- Источники шума

---

## Навигация

### Pathfinding

**Алгоритмы:**
- **A*** — поиск пути по сетке
- **NavMesh** — навигационная сетка
- **Waypoints** — заранее заданные точки

### Типы движения

| Тип | Описание |
|-----|----------|
| **Прямое** | К цели без обхода |
| **С обходом** | Обход препятствий |
| **По маршруту** | Заданный путь |

---

## Групповая тактика

| Тактика | Описание |
|---------|----------|
| **Окружение** | Атака с разных сторон |
| **Подавление** | Огонь на поражение |
| **Координация** | Совместные действия |
| **Отступление** | Организованный отход |
