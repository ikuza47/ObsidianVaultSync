> **Тип**: Игровой движок
> **Лицензия**: Open-source
> **Поддержка**: 2D и 3D

---

**Godot** — свободный и открытый игровой движок с поддержкой 2D и 3D. Использует собственный язык программирования GDScript (совместим с Python).

---

## Особенности

| Особенность | Описание |
|-------------|----------|
| **Открытый исходный код** | Полностью бесплатен, без роялти |
| **2D и 3D** | Поддержка обоих направлений |
| **GDScript** | Язык, похожий на Python |
| **Узловая система** | Всё в движке — узлы |
| **Кроссплатформенность** | Экспорт на ПК, мобильные, веб |

---

## Язык GDScript

**GDScript** — высокоуровневый язык программирования, разработанный специально для Godot.

**Преимущества:**
- Простой синтаксис (похож на Python)
- Интеграция с редактором
- Оптимизирован для игровых задач

---

## Применение

Используется для создания инди-игр, прототипов, образовательных проектов.
