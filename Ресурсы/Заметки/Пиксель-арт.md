> [!SUMMARY] Кратко
> **Определение**: Форма цифрового искусства, создаваемая посимвольно
> **Применение**: Спрайты, тайлы, анимации

---

**Пиксель-арт** — форма цифрового искусства, в которой изображения создаются и редактируются на уровне отдельных пикселей. Характеризуется ограниченной цветовой палитрой и стилизованной эстетикой.

---

## Основные понятия

| Термин | Определение |
|--------|-------------|
| **Спрайт** | Отдельное изображение объекта |
| **Тайл** | Базовый элемент для построения уровня |
| **Палитра** | Набор используемых цветов |
| **Dithering** | Техника смешивания цветов через шум |
| **Anti-aliasing (AA)** | Сглаживание краёв |
| **Pixel Perfect** | Без искажений при масштабировании |

---

## Разрешения

| Объект | Разрешение | Описание |
|--------|------------|----------|
| **Персонаж (малый)** | 16×16 | Классика (GBC, ранние игры) |
| **Персонаж (средний)** | 32×32 | Стандарт (GBA, DS) |
| **Персонаж (крупный)** | 64×64 | Детализация (современные инди) |
| **Тайл фона** | 16×16 / 32×32 | Зависит от масштаба |

---

## Цветовые палитры

### Популярные палитры

| Палитра | Цветов | Описание |
|---------|--------|----------|
| **PICO-8** | 16 | Ограниченная, ретро |
| **GBA** | 32 | Классическая |
| **Endesga 32** | 32 | Универсальная |

### Принципы выбора

- **Базовые цвета** (5-8) — тени, средние тона, света
- **Акцентные цвета** (3-5) — красный, зелёный, синий, жёлтый
- **Специальные цвета** (2-4) — фиолетовый, оранжевый, белый

---

## Техники рисования

### Lineart (Контур)

Создание контуров с соблюдением pixel perfect — все пиксели соединены, без разрывов.

### Shading (Затенение)

| Тип | Описание |
|-----|----------|
| **Cell shading** | Резкие переходы |
| **Gradient** | Плавные переходы |
| **Dithering** | Шум для сглаживания |

### Anti-aliasing (AA)

Сглаживание краёв через добавление промежуточных пикселей.

## Анимация

### Принципы

| Принцип | Описание |
|---------|----------|
| **Timing** | Время кадров |
| **Squash & Stretch** | Сжатие и растяжение |
| **Anticipation** | Подготовка к действию |
| **Follow Through** | Инерция после действия |

### FPS для анимаций

| Тип | FPS |
|-----|-----|
| **Бег/ходьба** | 12-15 |
| **Атака** | 20-24 |
| **Idle (покой)** | 6-8 |
| **Урон/смерть** | 24-30 |

## Ошибки новичков

| Ошибка | Описание |
|--------|----------|
| **Jaggies** | Неровные линии |
| **Banding** | Полосы при затенении |
| **Doubles** | Случайные пиксели |
| **Orphan pixels** | Одинокие пиксели |
| **Too many colors** | Слишком много цветов |
