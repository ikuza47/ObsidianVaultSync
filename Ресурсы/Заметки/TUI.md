> [!SUMMARY] Кратко
> **Определение**: Интерфейс в терминале
> **Элементы**: Символы, текст, рамки

---

**TUI** (Textual User Interface) — текстовый пользовательский интерфейс, который работает в терминале. Использует символы и текстовые компоненты для представления информации и получения ввода.

---

## Элементы TUI

| Элемент | Описание | Пример |
|---------|----------|--------|
| **Текст** | Основной способ отображения | Команды, вывод |
| **Символы** | Графика из символов | Рамки, иконки |
| **Рамки** | Разделение зон | ┌───┐ │Текст│ └───┘ |
| **Меню** | Списки опций | [1] Опция 1 |
| **Поля ввода** | Ввод текста | > _ |

---

## Преимущества

- Работает в любом терминале
- Низкие требования к ресурсам
- Быстрая работа
- Подходит для удалённого доступа

---

## Применение в играх

| Тип | Описание |
|-----|----------|
| **Терминал** | Интерфейс в стиле консоли |
| **Диалоги** | Текстовые диалоги с NPC |
| **Инвентарь** | Списки предметов |
| **Карты** | Символьные карты |
