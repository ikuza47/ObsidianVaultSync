> [!SUMMARY] Кратко
> **Определение**: Система пассивных бонусов в рогаликах
> **Типы**: Линейные, убывающие, уникальные
> **Применение**: Билдостроение, прогрессия

---

**Карточки навыков** — система пассивных бонусов в играх (особенно рогаликах), где игрок собирает карточки с эффектами, формируя уникальный билд.

---

## Типы эффектов

### Линейные бонусы

Эффекты складываются без убывания. Каждая карточка добавляет фиксированный бонус.

**Где использовать:** урон оружия, скорострельность, ёмкость магазина, скорость бега (с лимитом).

### Убывающая эффективность (Diminishing Returns)

Каждое следующее получение даёт меньше бонуса. Предотвращает чрезмерное усиление.

**Где использовать:** сопротивление урону, регенерация здоровья, шанс крита, снижение шума.

### Уникальные эффекты

Не стакируются, работает только одна карточка такого типа.

**Где использовать:** особые механики, воскрешение, изменение геймплея.

## Редкость

| Редкость | Шанс | Описание |
|----------|------|----------|
| **Обычная** | Высокий | Базовые бонусы |
| **Редкая** | Средний | Усиленные бонусы |
| **Эпическая** | Низкий | Мощные эффекты |
| **Легендарная** | Очень низкий | Уникальные механики |
| **Уникальная** | Минимальный | Меняют геймплей |

---

## Система слотов

### Ограничения

- **Старт** — начальное количество слотов
- **Расширение** — возможность увеличить количество слотов
- **Максимум** — предельное количество слотов

### Управление

- **Подобрать** — карточка занимает свободный слот
- **Выбросить** — удаление карточки
- **Заменить** — обмен старой карточки на новую

## Синергии

### Типы синергий

| Тип | Описание |
|-----|----------|
| **Аддитивная** | Эффекты складываются |
| **Мультипликативная** | Эффекты умножаются |
| **Кондициональная** | Работает при условии |
| **Триггерная** | Срабатывает на событие |

## Баланс

### Принципы

| Принцип | Описание |
|---------|----------|
| **Сравнимая ценность** | Цена соответствует эффекту |
| **Убывающая эффективность** | Для защитных и реген-эффектов |
| **Противовесы** | Сильные карточки имеют недостатки |
| **Ситуативность** | Некоторые карточки сильны только в ситуации |
