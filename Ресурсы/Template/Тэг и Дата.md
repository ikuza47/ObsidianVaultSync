> [!SUMMARY] Кратко
> **Назначение**: Базовый шаблон с параметрами
> **Применение**: Для заметок общего назначения

---

```markdown
---

дата::
теги::
конфликт::
связь::
движок::
```

---

## Параметры

| Параметр | Описание | Статус |
|----------|----------|--------|
| **дата** | Дата создания заметки | Обязательно |
| **теги** | Ключевые слова для категоризации | Необязательно |
| **конфликт** | Конфликтующие идеи | Необязательно |
| **связь** | Связанные файлы | Необязательно |
| **движок** | Игровой движок проекта | Необязательно |

---

## Пример использования

```markdown
---

дата:: [[04.03.2026 |04 мар. 2026]]
теги:: #заметка #геймдизайн
конфликт:: [[GUI]]
связь:: [[Проект/Игра/Главный]]
движок:: [[Godot]]
```
