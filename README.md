Этот репозиторий не представляет особой ценности практически для всех, потому что в нём находится только писанина моя, и всё.
Может быть в будущем из него выйдет хорошая база данных  (речь про [Заметки](./Ресурсы/Заметки) и [Материалы](./Ресурсы/Материалы)), т.к. я добавляю в них все определения/материалы и т.п. которые как-то используются в моих проектах. Короче вещь вроде и полезная, но довольно таки специализированная.
В целом можете использовать этот  репозиторий как базу под свой Obsidian, удалив не нужные вам файлы
